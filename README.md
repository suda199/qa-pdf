# わかんねぇとこがわかんねぇツール（通称：わかわかツール）

## 📌 概要 (Overview)
- ベースは中谷さんが作成した質問収集ツールです。
- 研修中の「質問をしづらい」「時間がなく質問できない」といった悩みを解消するため、リアルタイムで質問を「匿名」で投稿できるツールです。
- 「どんな質問があったのかをすべてメモするのが難しい」という講師側の手間を省けるようにしました。

## ✨ 主な機能 (Features)
- 操作パネルから共有するテキストを選択
- テキストの不明点にマーカーを付けて、質問を匿名で投稿
- 集めた質問をJSON形式で保存可能

## 🚀 使い方 (Usage)
> 💡 **前提：** 講師も受講者も同じ画面を共有して使用します。

1. **テキストの共有**
   - 講師は操作パネルからテキストを選択し、受講者に共有します。
   - ※ 須田はRenderにプロジェクトをデプロイしてURLを共有しています。
2. **質問の投稿（受講者側）**
   - 受講者はテキスト内の不明点をクリックし、マーカーを登録します。
   - マーカーの登録時に質問内容（「とりあえず分からない！」でもOK）を入力し、「マークを確定」ボタンを押下します。
3. **質問の確認とステータス管理**
   - 登録された質問は「操作パネル」下の「マーク一覧」に表示されます。
     - ⚠️ デフォルトのブラウザサイズだと登録された質問が表示されない場合があります。画面サイズを50%に縮小するとちょうどよく表示されます。
   - 投稿された質問には「解決済みにする」「未解決に戻す」ボタンがあり、状況に合わせてステータスを変化させられます。
4. **データの保存**
   - 集めた質問は、最後にJSONファイルにまとめて保存可能です。

### ℹ️ 補足仕様
Render無料版でデプロイしている関係上、誰も触れていない状態が15分続くとセッションが切れてしまいます。そのため、対策として**10分おきに再レンダリング（スリープ防止）するシステム**を組み込んでいます。

## 🛠 インストール・セットアップ (Installation)
ローカル環境で動かすための手順です。

### 1. リポジトリのクローン
```bash
git clone [https://github.com/suda199/qa-pdf.git](https://github.com/suda199/qa-pdf.git)
```

### 2. ディレクトリの移動
```bash
cd qa-pdf
```

### 3. 依存パッケージのインストール
```bash
npm install
```

### 4. サーバーの起動
```bash
npm run start
```
