## わかんねぇとこがわかんねぇツール(通称：わかわかツール)

## 📌 概要 (Overview)
・ベースは中谷さんが作成した質問収集ツール
・研修中の「質問をしづらい」「時間がなく質問できない」といった悩み解消のため、リアルタイムで質問を「匿名」で投稿できるツール。
・どんな質問があったのか、全部メモするのが難しいのでその手間を省けるようにした。

## ✨ 主な機能 (Features)
・操作パネルから共有するテキストを選択
・テキストの不明点にマーカーを付けて質問を匿名で投稿
・集めた質問はJSONで保存可能

## 🚀 使い方 (Usage)
前提：講師も受講者も同じ画面を共有。
・テキストを選択し、受講者に共有。
　→須田はRenderにプロジェクトをデプロイしてURLを共有
・受講者はテキスト内の不明点をクリックし、マーカーを登録
・マーカーの登録時に質問内容(とりあえず分からない！でもOK)を入力し、「マークを確定」ボタンを押下
・「マークを確定」ボタンを押下により登録された質問は「操作パネル」下の「マーク一覧」に表示。
　→デフォルトのブラウザサイズだと、登録された質問が表示されない。画面サイズを50%にするとちょうどよくなる。
・投稿された質問に「解決済みにする」「未解決に戻す」ボタンを設定。状況に合わせて変化させる。
・集めた質問はJSONファイルにまとめて保存可能。

※Render無料版でデプロイしている関係上、誰も触ってなければ15分でセッションが切れてしまうので、10分おきに再レンダリングするシステムが入っています。

## 🛠 インストール・セットアップ (Installation)
ローカル環境で動かすための手順です。
# リポジトリのクローン
git clone [https://github.com/suda199/qa-pdf.git](https://github.com/suda199/qa-pdf.git)

# ディレクトリの移動
cd qa-pdf

# 依存パッケージのインストール（※使っている環境に合わせて書き換えてください）
npm install 

# サーバーの起動
npm run start
